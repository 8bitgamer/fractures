Usage

  • For the typescale to work properly, you must define font-size and line-height on body.
  • Dimension blocks are built as 16px (temporarily) — you can overwrite this in src/variables and build your version with gulp build.

Install

  • npm install fractures
  • git clone https://github.com/krisztianpuska/fractures.git
  • download vanilla → fractures.min.css ({{ meta.size }} kB gzipped)
  • download prefixed for latest 2 browser version → fractures.prefixed.min.css
