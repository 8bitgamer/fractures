Usage

For the typescale to work properly, you must define font-size and line-height on body.
16px/1.5 should work on most cases just fine.

Dimension blocks are locked to 16px (for now) — you can overwrite settings in src/variables and build your version with gulp build.

Install

  • npm install fractures
  • git clone https://github.com/krisztianpuska/fractures.git
  • download vanilla → fractures.min.css (2.16 kB gzipped)
  • download prefixed for latest 2 browser version → fractures.prefixed.min.css (2.22 kB gzipped)
